ASIDE.IO Is (Sorta) Shutting Down

But It Will Live On!

The ASIDE.IO website in it's current form is shutting down 6/19/2020 at 11:59 PM CST.

However, that doesn't mean ASIDE is going away. ASIDE is now open source, and you can view it's code here. Additionally, it's now very easy to setup your own instance of ASIDE. Follow the steps here to create your personal instance of ASIDE that you can host for free and keep up as long as you want.

Beyond 6/19/2020 you will not be able to visit www.aside.io to use ASIDE. You will need to create your own instance of ASIDE using the steps at the link above if you want to continue using ASIDE. After that date it will redirect to ASIDE's GitHub repository. This is because I will no longer be able to host the ASIDE.IO website.

However, just because I can no longer host the website doesn't mean that you can't. If you see the value in this tool and are seriously interested in hosting ASIDE for the developer community, please contact me at admin@aside.io and I can potentially hand the ASIDE.IO domain over to you to keep the website alive.

Thanks for your support and my apologies for the short notice.
